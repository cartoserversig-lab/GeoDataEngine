# Roadmap

## Version 1 — en cours

- Acquisition automatique des données (sources IGN)
- Harmonisation géographique (contrôle des SCR, reprojection automatique, normalisation)
- Génération automatique d'un environnement SIG (base GeoPackage, arborescence projet, catalogue de données, métadonnées)

### Critères de réussite de la V1

- [ ] une emprise utilisateur peut être importée
- [ ] les données nécessaires sont téléchargées automatiquement
- [ ] les données sont reprojetées automatiquement
- [ ] une base GeoPackage structurée est générée
- [ ] un rapport de traitement est produit

### Sources de données V1

Orthophotos, Lidar HD, MNT, MNS, BD TOPO, bâtiments, hydrographie, réseaux routiers (IGN).

## Version 2

Génération automatique de produits dérivés : pente, exposition, ombrage, courbes de niveau.

## Version 3

Préparation automatique de jumeaux numériques : bâtiments 3D, scènes 3D, modèles de terrain.

## Version 4

Diffusion : ArcGIS Online, services web, API, portail territorial.

## Sources de données futures

OpenStreetMap, Copernicus, Sentinel, Cadastre, INSEE, RPG, données locales.
